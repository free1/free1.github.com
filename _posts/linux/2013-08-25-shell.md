## 基本语法
* `$`:  访问变量
* `echo`:  输出 
* `[或test`:  布尔判断
* `#`:  单行注释
* `function fname() {xx}`:  定义函数
* if条件判断:

```
if [ boolean ]; then
elif [ boolean ]; then
else
fi
```
* case条件判断

```
case  $xx in
  "aa")
    echo "aa"
    ;;
  "bb")
    echo "bb"
    ;;
  *)
    echo "other"
    exit 1
    ;;
esac
```
* 字符串截取: `echo ${STRING:$POS:$LEN}`
