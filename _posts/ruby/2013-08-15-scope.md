* 全局变量: 可以在任何作用域中访问，尽量少使用。   
* 顶级实例变量: 可以代替全局变量，但更加安全。   

## 共享作用域   

```
my_var = "Success"

class MyClass
    # 需要打印my_var...

    def my_method
        # 需要打印my_var...
    end
end
```

重写后可以完成:   

```
my_var = "Success"

MyClass = Class.new do
    puts "#{my_var}"

    define_method :my_method do
        puts "#{my_var}"
    end
end

MyClass.new.my_method

```
